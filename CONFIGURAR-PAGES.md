# Configurar GitHub Pages — passo a passo

## 1. Abra o repositório

Acesse: https://github.com/Vincs02/wellington-limeira.github.io

## 2. Vá em Settings

Clique em **Settings** (Configurações) no menu do repositório.

## 3. Abra Pages

No menu lateral esquerdo, clique em **Pages** (abaixo de "Code and automation").

## 4. Configure o Source

Em **Build and deployment**:

- **Source**: escolha **Deploy from a branch** (NÃO use GitHub Actions)
- **Branch**: selecione **main**
- **Folder**: selecione **/ (root)**
- Clique em **Save**

## 5. Aguarde 1–2 minutos

O site ficará disponível em: **https://vincs02.github.io/wellington-limeira.github.io/**
